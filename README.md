# Soundr

#### A music player with following functionalities

- Register user
- Login user
- Player
  - Playing songs
  - Timeline change
  - Volume change
  - Making song loop
- Upload song with values: title, author, song file and cover image
- Creating playlist
  - Adding songs to playlists
  - Deleting songs from playlists
